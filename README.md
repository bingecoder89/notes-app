# 📝 Everynote

A modern notes management application built with React that helps users create, organize, archive, and summarize notes efficiently.

## 🚀 Live Demo

🔗 Live App: https://everynote11.netlify.app/

🔗 GitHub Repository: https://github.com/bingecoder89/notes-app.git

---

## ✨ Features

### Notes Management

- Create notes
- Edit notes
- Delete notes
- Archive and unarchive notes
- Pin important notes
- Tag notes for organization

### Search & Filtering

- Real-time note search
- Filter notes by tags
- Search term highlighting

### User Experience

- Mobile responsive design
- Dark mode support
- Toast notifications
- Empty state UI
- Confirmation dialog before deletion
- Markdown rendering support

### AI Features

- AI-powered note summarization
- Integrated Google Gemini API
- Loading and error state handling
- One-click note summaries

### Persistence

- Local Storage support
- Notes persist across browser refreshes
- Archive state persistence
- Pinned notes persistence

## ⚙️ Installation

Clone the repository:

```bash
git clone <your-repository-url>
```

Navigate to the project:

```bash
cd everynote
```

Install dependencies:

```bash
npm install
```

Create a `.env` file:

```env
VITE_GEMINI_API_KEY=your_api_key_here
```

Start the development server:

```bash
npm run dev
```
